Traceroute:
for linux / macos command : traceroute ( ip or address)
for windows command : tracert ( ip or address)

Alternative of traceroute: mtr for linux/ macOS ; pathping for windows.

(When the TTL field reaches zero, the packet is discarded and an ICMP Time Exceeded message is sent back to the originating host. knowing this thing is important to understand traceroute)

Traceroute's working method: Traceroute uses the TTL field by first setting it to one for the first packet, then two for the second, three for the third and so on. By doing this clever little action, traceroute makes sure that the very first packet sent will be discarded by the first router hop. This results in an ICMP Time Exceeded message, the second packet will make it to the second router, the third will make it to the third, and so on. This continues until the packet finally makes it all the way to its destination. For each hop, traceroute will send three identical packets.
