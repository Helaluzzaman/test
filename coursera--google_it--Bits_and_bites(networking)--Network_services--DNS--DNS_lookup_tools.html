DNS lookup tools


1. nslookup


nslookup normally give A-record of a domain or ip.
Interactive Mode: using the nslookup command with no options, the command provides a prompt to enter additional information in what is called an interactive mode.

But we can change the resource type by the command:
set type=MX
set type=AAAA
set type=TXT
set type=A
set type=Cname
set type=SRV
etc
set debug (will display full response packet)
