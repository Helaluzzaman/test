Transport Layer


Transport layer's work is done with multiplexer and demultiplexer.


Port : A 16 bit number that's used to direct traffic to specific services running on a networked computer.
TCP : it is a connection oriented protocol
A connection-oriented protocol is one that establishes a connection, and uses this to ensure that all data has been properly transmitted.
UDP (User datagram Protocol) connection less protocol. ( It is usually used in video steaming and etc)
