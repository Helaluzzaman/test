ethernet frame


Data packet at ethernet level is known as ethernet frame.
CRC- Cyclical Redundency Check
Vlan - virtual lan (For making multiple lan within a single physical Lan)
Last byte of the Preamble is known as SFD(Start Frame Dilimeter).
SFD (start frame delimeter) indicates that the preamble is over and the actual frame contents will now follow.
This frame is usually limited in size to 1518 bytes.
FCS - Frame Checksum sequence
FCS (32 bites ) represents the checksum value of the entire frame. In the FCS , CRC is used.
