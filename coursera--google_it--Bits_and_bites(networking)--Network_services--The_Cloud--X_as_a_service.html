X as a service:
Cloud can be described as infrastructure as a Service.

IaaS> PaaS> SaaS

(Infrastructure as a service abstracts away the physical infrastructure you need and platform as a service abstracts away the server instances you need.)

IaaS: You should not have to worry about building your own network or your own servers.

PaaS: Platform as a service is a subset of cloud computing where a platform is provided for customer to run their service.
(This basically means that an execution engine is provided for whatever software someone wants to run.)

SaaS: Software as a service is essentially a way of licensing the use of software to others while keeping that software centrally hosted and managed.
such as: gmail for bussiness, office 365 outlook.

For the application of SaaS, web browsers have become so feature packed.
