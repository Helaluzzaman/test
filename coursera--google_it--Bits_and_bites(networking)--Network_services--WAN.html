WAN


WAN :A wide area network acts like a single network but spans across multiple physical locations.
Local loop: In a WAN, the area between a demarcation point and the ISP's core network is known as local loop


Supplement: Unless you end up working as a network engineer at an Internet service provider, chances are you’ll never have to directly interact with any of the protocols that allow for WAN technologies to work. But, if you’d like to learn more about some of the more popular options, you can read about them on Wikipedia. It’s important to note that these protocols usually define how both the physical and data link layers need to operate when they’re deployed.
