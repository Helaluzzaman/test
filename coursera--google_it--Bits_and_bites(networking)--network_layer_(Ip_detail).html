network layer (Ip detail)


As there are billions of Mac address in the internet and they are not assembled in order so it is impossible to figure out an unique mac address to deliver data. So network layer(ip addressing ) is important.
ip belongs to network not device.
DHCP stands for Dynamic Host Configuration Protocal.

Packet for ip is called ip datagram (like ethernet frame in data link layer)
Ip class A - first octate's first bit is 0 fixed .( 0 - 128)
Ip class B - first octate's first two bit are 10 fixed.( 128 -192)
Ip class C - first octate's first three bits are 110 fixed.(192 - 223)
