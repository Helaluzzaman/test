Proxy service


A proxy service is a server that acts on behalf of a client in order to access another service.

It provides anonymity, security, content filtering, increase performance

Note : A gateway definitely meets the definition of a proxy and how it works.

Proxy server: proxies are any server that act as a intermediary between a client and another server.

Reverse proxy server: A reverse proxy is a service that might appear to be a single server to external clients, but actually represents many servers living behind it.
